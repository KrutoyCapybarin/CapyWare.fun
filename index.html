CapyWare

External FFlag executor for Roblox. Writes client-side feature flags directly to memory — no DLL injection, no hooks. Full scripting engine included.

↓ Download GitHub →

Installation

Первый запуск — автоматический установщик. Просто скачай и запусти.

Первый запуск

При запуске CapyWare.exe с рабочего стола или папки загрузок:

  1. Создаётся папка %DESKTOP%/CapyWare/ с подпапками
  2. Exe копируется в папку
  3. Создаётся ярлык CapyWare.lnk на рабочем столе
  4. Запускается установленная копия
ℹ
После установки используй ярлык на рабочем столе или exe из папки CapyWare.

Зависимости (устанавливаются автоматически)

PyQt5     — UI
keyboard  — глобальные хоткеи
requests  — HTTP для request()

Folder Structure

CapyWare/ ├── autoexec/ ← скрипты запускаются при каждом Inject ├── scripts/ ← библиотека скриптов (Open/Save) ├── workspace/ ← рабочая папка для readfile/writefile ├── logs/ ← логи сессий: cw_YYYY-MM-DD.log ├── CapyWare.exe └── config.cfg ← настройки (JSON, сохраняются автоматически)
ℹ
autoexec/ — все файлы .cw .ff .txt запускаются в алфавитном порядке при нажатии ⚡ Inject.
ℹ
logs/ — каждая сессия пишет в cw_YYYY-MM-DD.log. Все события: инжект, скрипты, флаги, ошибки.

FFlags ★

Основная функция CapyWare — запись FFlag значений в память Roblox. Два формата синтаксиса.

Key = Value

-- числовой флаг
TaskSchedulerTargetFps = 9999

-- булев флаг
DebugDisplayFPS = true
DisableDPIScale = false

-- значения true/false/1/0 — все работают
DebugDisplayFPS = 1
ℹ
Заглавная буква в начале имени = FFlag. Строчная буква = переменная скрипта.

JSON Mode

Если текст начинается с { — автоматически парсится как JSON.

{
  "TaskSchedulerTargetFps": 9999,
  "DebugDisplayFPS": true,
  "DisableDPIScale": true
}

Prefix Stripping

Если флаг не найден в дампе — CapyWare отрезает известный префикс и ищет bare-имя.

Что написалЧто ищется
DFIntTaskSchedulerTargetFps→ TaskSchedulerTargetFps
FFlagDebugDisplayFPS→ DebugDisplayFPS
DFLogAbuseReport→ AbuseReport

Поддерживаемые префиксы: DFInt DFFlag DFLog DFString FInt FFlag FLog FString SFInt SFFlag SFLog SFString

Dump System

RING 0Основной дамп (default: GitHub). Ищется первым.
RING 1Резервный. Используется если флаг не найден в ring 0.
RING 2+Дополнительные дампы. Количество не ограничено.
adddump("https://example.com/dump.h", "backup", "auto")
usedump("backup", 1)
getdumps()

Functions

Обычные функции — объявляются через function и вызываются по имени.

function boost(fps)
    TaskSchedulerTargetFps = fps
    print("fps set to " .. fps)
end

boost(9999)
boost(60)

capyware.* Functions

Встроенные функции ядра, доступные через префикс capyware.. Объявляются в конце CapyWare_core.py через add_cw_function.

Объявление (в core.py)

# в конце CapyWare_core.py
add_cw_function(desync(tof)
    if tof == true then
        PhysicsSenderMaxBandwidthBps = 0
    else
        PhysicsSenderMaxBandwidthBps = 1000
    end
end)

Вызов (в редакторе капивара)

capyware.desync(true)
capyware.desync(false)
ℹ
Функции зарегистрированные через add_cw_function доступны только через capyware. префикс. Обычный вызов по имени не работает.

capyware.exec()

-- выполнить Python код с доступом к core
capyware.exec("core.on_log('[py] hello')")
capyware.exec("import os; os.startfile('notepad.exe')")

Доступные объекты: core, os, time, re

Events

Ивенты живут до ⏏ Eject и срабатывают автоматически.

OnKeyPress

OnKeyPress("q"):
    TaskSchedulerTargetFps = 9999
end

OnKeyPress("e"):
    TaskSchedulerTargetFps = 60
end
ℹ
Буквы — строчные. Спецклавиши: end insert delete home pageup pagedown f1–f12 leftshift rightalt и др.

OnInject / OnEject

OnInject():
    print("injected!")
    TaskSchedulerTargetFps = 9999
end

OnEject():
    print("bye")
end

capyware.* API

capyware.inject()
Подключиться к Roblox. Аналог кнопки ⚡.
capyware.eject()
Eject — освободить хендл, очистить хоткеи. Аналог кнопки ⏏.
capyware.offsets()
Перезагрузить ring 0 дамп с GitHub.
capyware.undoall()
Откатить все изменённые флаги к исходным значениям.
capyware.notify("msg")
Вывести сообщение в консоль с префиксом 🔔.
capyware.attached()
Вывести true/false — подключён ли к Roblox.
capyware.version()
Вывести текущую версию CapyWare.
capyware.lag(seconds)
Lagswitch — блокировать сеть Roblox на N секунд через netsh.
capyware.desync(bool)
Включить/выключить десинк (PhysicsSenderMaxBandwidthBps).
capyware.clear_globals()
Очистить все глобальные переменные.
capyware.exec("python code")
Выполнить произвольный Python код с доступом к core.
capyware.set("key", value)
Записать значение в capyware-переменную.
capyware.get("key")
Прочитать значение capyware-переменной в консоль.
