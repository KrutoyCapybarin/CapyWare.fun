CapyWare

External FFlag executor for Roblox. Modify client-side feature flags directly from memory — no injection, no DLL. Built with Python, runs from a clean GUI.

↓ Download V1.0 GitHub →

Installation

First-time setup is fully automatic. Just download and run.

First Run (from Desktop or Downloads)

When you run CapyWare.exe from Desktop or Downloads for the first time, the installer kicks in automatically:

  1. Creates %DESKTOP%/CapyWare/ with subfolders
  2. Copies CapyWare.exe into the new folder
  3. Creates a shortcut CapyWare.lnk on your Desktop
  4. Launches the installed copy
  5. Deletes the original download — cleanup done
ℹ
After install, always use the Desktop shortcut or the exe inside the CapyWare folder. The downloaded file deletes itself.

Dependencies

The launcher installs everything automatically on first run via pip:

PyQt5     — UI framework
keyboard  — global hotkeys
requests  — HTTP (request() scripting function)

Folder Structure

CapyWare/ ├── autoexec/ ← scripts run automatically on Inject │ └── boost.cw ├── scripts/ ← your saved scripts, Open/Save goes here │ └── fps_unlock.cw ├── workspace/ ← readfile/writefile default directory │ └── log.txt ├── CapyWare.exe ← main executable └── config.cfg ← auto-saved settings (JSON)
ℹ
autoexec/ runs all .cw, .ff, .txt files in alphabetical order every time you click ⚡ Inject. Place your persistent scripts here.

FFlags

The core purpose of CapyWare. Two syntax formats are supported.

Key = Value syntax

-- set target FPS
TaskSchedulerTargetFps = 9999

-- enable debug FPS counter
DebugDisplayFPS = true

-- disable DPI scaling
DisableDPIScale = true

-- set physics bandwidth
PhysicsSenderMaxBandwidthBps = 100000000

JSON syntax

{
  "TaskSchedulerTargetFps": 9999,
  "DebugDisplayFPS": true,
  "DisableDPIScale": true,
  "PhysicsSenderMaxBandwidthBps": 100000000
}
ℹ
Auto JSON mode detects { at the start and switches parser automatically. Can be disabled in Settings.

Prefix Stripping

If a flag isn't found in the dump, CapyWare strips known prefixes and tries the bare name. Works across all dump rings.

InputTries
DFIntTaskSchedulerTargetFps→ TaskSchedulerTargetFps
FFlagDebugDisplayFPS→ DebugDisplayFPS
DFLogAbuseReport→ AbuseReport

Supported prefixes: DFInt DFFlag DFLog DFString FInt FFlag FLog FString SFInt SFFlag SFLog SFString

Scripting Engine

Enable Scripting Engine in Settings to unlock automation functions. Flags still work exactly the same — scripting is additive.

wait()

TaskSchedulerTargetFps = 9999
wait(2)
print("2 seconds later")
DebugDisplayFPS = true

print()

print("Hello from CapyWare")
print("flags applied!")

json()

-- inline JSON block inside a script
json({"DebugDisplayFPS": true, "TaskSchedulerTargetFps": 9999})

getexecutorname() / identifyexecutor()

getexecutorname()    -- prints "CapyWare V1.0"
identifyexecutor()   -- same

Events

Events register handlers that fire automatically. They persist until ⏏ Eject.

OnKeyPress

OnKeyPress("Q"):
    TaskSchedulerTargetFps = 9999
    print("fps unlocked")
end

OnKeyPress("E"):
    TaskSchedulerTargetFps = 60
    print("fps locked")
end

OnAttach

-- fires when ⚡ Inject succeeds
OnAttach():
    print("attached to Roblox!")
    TaskSchedulerTargetFps = 9999
end

OnEject

-- fires when ⏏ Eject is clicked
OnEject():
    print("detached")
end
⚠
OnKeyPress handlers use the global keyboard hook — they fire even when CapyWare is minimized. All handlers are cleared on Eject.

Filesystem

All paths are relative to the CapyWare folder unless absolute. Requires Executor Mode enabled in Settings.

readfile("path")
Returns file contents as string. Prints to console.
writefile("path", "data")
Write data to file. Creates if not exists, overwrites if exists.
appendfile("path", "data")
Append data to end of file.
delfile("path")
Delete a file.
isfile("path")
Returns true/false. Prints result to console.
isfolder("path")
Returns true/false. Prints result to console.
makefolder("path")
Create a folder (and parents if needed).
delfolder("path")
Delete folder and all contents recursively.
listfiles("path")
List all files/folders in directory. Prints to console.
loadfile("path")
Read and execute a script file. Runs in a new ScriptRunner thread.

Example

-- log applied flags to file
writefile("workspace/session.log", "session started")

-- load a preset script
loadfile("scripts/fps_boost.cw")

-- check if backup exists
isfile("workspace/backup.txt")

Multi-Ring Dump System

CapyWare searches for flags across multiple dump sources in priority order. Ring 0 is always searched first.

RING 0 → Primary dump (default: GitHub current_dump.h). Searched first.
RING 1 → Fallback. Used if flag not found in ring 0.
RING 2 → Second fallback. Any number of rings supported.

adddump()

-- add a .h dump (auto-detect format)
adddump("https://example.com/dump.h", "backup", "auto")

-- add a JSON dump explicitly
adddump("https://example.com/flags.json", "json_source", "json")

-- ring assigned automatically (next after highest)

usedump()

-- move a dump to a specific ring
usedump("backup", 1)

getdumps()

-- list all loaded dumps and their rings
getdumps()

HTTP / request()

Send HTTP requests from scripts. Useful for Discord webhooks, logging, or any external API.

-- simple GET
request({"url": "https://httpbin.org/get", "method": "GET"})

-- Discord webhook
request({
  "url": "https://discord.com/api/webhooks/YOUR_WEBHOOK",
  "method": "POST",
  "headers": {"Content-Type": "application/json"},
  "body": "{\"content\": \"CapyWare injected\"}"
})
FieldTypeRequiredDescription
urlstring✓Target URL
methodstring—HTTP method. Default: GET
bodystring—Request body
headersobject—Additional request headers

capyware.* API

capyware.offsets()
Reload ring 0 dump from GitHub.
capyware.eject()
Eject — release handle, clear all key handlers. Same as clicking ⏏.
capyware.undoall()
Restore all modified flags to their original values using undo log.
capyware.notify("msg")
Print a message to the output console with 🔔 prefix.
capyware.attached()
Prints true/false — whether currently attached to Roblox.
capyware.version()
Prints current CapyWare version string.

Example usage

OnAttach():
    capyware.notify("attached to Roblox")
    capyware.offsets()     -- refresh dump on attach
    loadfile("scripts/main.cw")
end

OnKeyPress("F1"):
    capyware.undoall()     -- reset all flags
    print("flags restored")
end
